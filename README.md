# Project Banner

```markdown
## 🌾 About AGRIVISION

AGRIVISION is an AI-powered agricultural intelligence platform designed to assist farmers in making data-driven decisions.

By leveraging Machine Learning, soil analytics, weather conditions, and crop-related datasets, the system predicts crop yield and recommends the most suitable crops for cultivation, helping farmers maximize productivity and profitability while promoting sustainable agriculture.
```

````
# Installation

```markdown
## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/umme2323hani/Major_Project-Agrivision.git
cd Major_Project-Agrivision
````

### Install Dependencies

```bash
pip install flask pandas numpy scikit-learn
```

### Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
